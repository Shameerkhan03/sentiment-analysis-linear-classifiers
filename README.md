# Sentiment Classification (BoW + Linear Models)

Implementation of a sentiment analysis pipeline using bag-of-words features and classical linear classifiers (Perceptron, Averaged Perceptron, Pegasos).

The project focuses on building text representations from scratch and evaluating simple online learning algorithms for binary sentiment classification, with experiments in feature engineering and hyperparameter tuning to improve model performance.

Includes training, validation, and comparison of models on review datasets.
